## 概述

数值模拟技能负责对游戏中的数值模型进行定量验证。支持蒙特卡洛随机模拟（分析概率分布和极端情况）、确定性计算（验证固定输入下的输出）、以及敏感性分析（识别关键参数对结果的影响程度）。输出包括统计分布、极值分析和平衡性建议。

## 步骤

1. **解析模型** — 将自然语言描述的数值模型转化为可计算的公式，明确输入变量、中间变量和输出变量
2. **配置参数** — 定义每个参数的分布类型（均匀/正态/固定值）、取值范围和步长
3. **执行模拟** — 根据模拟类型运行计算：蒙特卡洛生成随机样本、确定性逐点计算、敏感性逐参数扫描
4. **统计分析** — 计算均值、中位数、标准差、P5/P95 分位数、最大/最小值
5. **异常检测** — 识别超出合理范围的极端值、分布偏斜、数值溢出等异常
6. **生成建议** — 基于分析结果给出参数调整和平衡性优化建议

## 示例

**输入**:
- model: "伤害 = 攻击力 × (1 + 暴击倍率 × 是否暴击) - 防御力 × 减伤系数"
- parameters: "攻击力[100-500], 暴击率[10%-30%], 暴击倍率[1.5-2.0], 防御力[50-200], 减伤系数[0.5-0.8]"
- simulation_type: monte_carlo

**输出**:

```markdown
# 数值模拟报告

## 模型
- **公式**: DMG = ATK × (1 + CRIT_MULT × IS_CRIT) - DEF × DEF_RATE
- **模拟类型**: monte_carlo
- **迭代次数**: 10000

## 参数设置
| 参数 | 分布类型 | 最小值 | 最大值 | 均值 |
|------|----------|--------|--------|------|
| ATK | 均匀分布 | 100 | 500 | 300 |
| CRIT_CHANCE | 均匀分布 | 10% | 30% | 20% |
| CRIT_MULT | 均匀分布 | 1.5 | 2.0 | 1.75 |
| DEF | 均匀分布 | 50 | 200 | 125 |
| DEF_RATE | 均匀分布 | 0.5 | 0.8 | 0.65 |

## 结果统计
| 指标 | 值 |
|------|----|
| 均值 | 237.5 |
| 中位数 | 220.3 |
| 标准差 | 145.2 |
| P5 | 12.0 |
| P95 | 510.8 |
| 最小值 | -60.0 |
| 最大值 | 840.0 |

## 异常分析
- **负伤害**: 约 3.2% 的情况下出现负伤害（低攻击 + 高防御），需设置伤害下限为 0 或 1
- **极高伤害**: P99 达到 680，高攻击暴击 + 低防御场景下伤害波动过大

## 平衡性建议
- 增加伤害下限保护（min=1），避免出现 0 伤或负伤
- 考虑收窄暴击倍率范围至 1.5-1.8，降低极端高伤害出现概率
- 防御减伤系数上限建议降至 0.7，防止高防目标过于坚挺
```
